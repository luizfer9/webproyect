A continuación os muestro un par de ejemplos para que veáis como funcionan los comandos CMD en la práctica:

“shutdown –r –f –t 5”: Reinicio de equipo tras 5 segundos de espera.

“net user “nombreusuario” /domain”: Muestra las propiedades de un usuario del dominio (ultimo cambio de contraseña, cuenta activa o no, grupos a los que pertenece…).

“nbtstat –a “ip_equipo””: Introduciendo la IP de un equipo puedes obtener el nombre de la maquina y la MAC

Otra forma de ejecutar los comandos CMD en Windows

Todos los comandos CMD anteriormente mencionados también los puedes ejecutar desde el cuadro de Inicio sin tener que abrir la consola de MS-DOS.
Para ello haz lo siguiente:
? Escribe CMD /K comando+ opción
Esto ejecutará el comando con la opción y dejara abierta la ventana de CMD.
? Escribe CMD /C comando+ opción
Esto ejecutará el comando con la opción y cerrara la ventana de CMD al terminar.
Dos ejemplos:
CMD /K IPCONFIG /ALL
CMD /C START http://google.com
